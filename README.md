# Tic Tac Toe — Built with OpenCode on Android

A clean, minimal, mobile-responsive Tic Tac Toe game built entirely on an Android phone using Termux, OpenCode, and React.

## The Goal

This repo is a proof of concept: can you set up a full AI coding harness on a fresh Android device and ship a working app? The answer is yes. Every line of code, every config file, and this README were written by an AI agent running in a terminal on Android.

## Features

- 2 Player local and vs AI game modes
- Dark minimal theme with animated X and O marks
- Fully mobile responsive with touch controls
- Smooth animations (pop-in marks, pulsing wins)
- Score tracking across rounds

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.
